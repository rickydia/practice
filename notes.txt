JSX
	non-native elements added to javascript
	eg,
		h1 = <h1>Hello world</h1>;
	a JSX element can be stored in a variable
	have to declare each elem as const???
	attributs can be given to JSX elements
	eg, 
		const p1 = <p id="large">foo</p>;
		const p2 = <p id="small">bar</p>;
